# **🎬 Movie Explorer – Movie API Web Application**

A responsive movie web application that allows users to search and explore movies using data fetched from a public Movie API. The project demonstrates API integration, dynamic content rendering, and responsive web design using HTML, CSS, and JavaScript.


## **📌 Project Overview**

This project was developed as part of the Dynamic Website Development with API Integration assignment. The application retrieves movie data from an external API and displays it dynamically on the website.

Users can search for movies, view details such as posters, ratings, and descriptions, and explore popular movies through an interactive and user-friendly interface.


## **🚀 Features**

🔍 Search movies by title

🎥 View movie posters and basic information

⭐ Display movie ratings and release dates

📄 View detailed movie descriptions

📱 Fully responsive design for desktop, tablet, and mobile

🌐 Dynamic data loading using JavaScript and API requests



## **🛠️ Technologies Used**

HTML5 – Website structure

CSS3 – Styling and responsive design

JavaScript – API integration and dynamic content

Movie API – Fetch movie data
